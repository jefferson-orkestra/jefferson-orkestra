### What I do

I help service businesses replace repetitive, judgment-free human work with **measurable, auditable automation**. Not "AI everywhere" — the right tool for each layer: deterministic pipelines where rules suffice, LLM reasoning where interpretation is needed.

Every system I ship answers three questions explicitly:
- **What runs autonomously?**
- **What escalates to a human?**
- **What does it measurably replace?**
